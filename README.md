# Phần Mềm Tổng Hợp Sửa Chữa

Web app tra cứu tồn kho và quản lý sửa chữa, kết nối Google Sheets.

**A product by Thuanmv**

## Chức năng

| Nút | Loại | Mô tả |
|-----|------|-------|
| Tra Tồn Kho | Tìm kiếm | Tìm cột C → trả về B, C, J |
| Scan BBKT 1/2/3 | Mở link | Mở liên kết biên bản kiểm tra |
| P.KT AT 1 / P.KTAT 2 / P.KTAT 4 | Tìm kiếm | Tìm cột E → trả về B, D, E, N, O, H (sheet: QUAN LY BBKT) |
| Dự phòng 1/2/3 | Mở link | Nút tùy chỉnh |
